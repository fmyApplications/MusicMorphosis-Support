Privacy Policy

MusicMorphosis does not collect, store, or share any personal information from its users.

We respect your privacy and are committed to ensuring that your data is secure. Since MusicMorphosis does not collect user data, there is no information to manage, store, or share.

MusicMorphosis uses the Discogs API to search for and display album information. The use of the Discogs API does not involve the collection or sharing of any user data. However, basic metadata (such as the app's user agent or IP address) may be transmitted to Discogs as part of standard API usage.
